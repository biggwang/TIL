## REST API란

### API URL 설계 방식에 대한 방법론이다.
- 클라이언트가 URL을 보고 어떤 동작을 하겠구나 명확하게 예상할 수 있다.
- 예상가능한(쉬운) URL로 다양한 클라이언트(웹, 안드로이드 등)를 대응하는 인터페이스(이것은 REST API가 아니라 API 자체가 그런것)
- 동작/리소스구조/id 형태

### RESTful 하다??
- 그만큼 URL을 보면 자원에 대한 액션이 어떨지 예상 가능 하다는 이야기이다.
