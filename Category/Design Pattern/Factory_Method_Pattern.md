## Template Method Pattern
상위클래스에서 골격을 만들고, 하위 클래스에서 구체적인 처리를 한다.

### 설명
- 상위클래스
    + 하위클래스에서 구현하지만 큰틀은 상위클래스에 templateMethod에 의해서 동작 됨
- 하위클래스
    + 상위에서 구현하지 않은 메소드 구현

### 장점
- 로직 공통화
- 수정은 templateMethod에서만 하면 된다.
