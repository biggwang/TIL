## 프로세스 커뮤니케이션 (IPC - Inter Process Communication)

### 프로세스A가 직접 프로세스B를 접근 할 수 없다.

### 근데 프로세스간 통신이 왜 필요할까??
- 요즘 CPU는 멀티코어
- 성능(속도)을 높이기 위해 여러 프로세스를 만들어서 동시 실행
- 이 때 프로세스간 상태 확인 및 데이터 송수신이 필요 한것이다.


### 실제 프로세스에서 커널 공간(물리메모리)은 공유한다. (리눅스 예)
즉 IPC, 프로세스간 통신하는 방법은 바로 이 커널 공간을 이용하여 통신한다는 의미

### IPC 기법
- Message Queue
- Shared Memory
- Pipe
- Signal
- Semaphore
- Socket
