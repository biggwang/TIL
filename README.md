# :lock: 작성규칙
- 이해한 내용을 **나만의 언어** 로 작성
- 각 doc별로 Learning-Queue.md 파일을 만들어 궁금한거 배우고 싶은거 queue 정리
- 못한것은 주말에 **반드시** 마무리
- 제목에 반드시 TAG 부터 붙인다.

# Tag 작성규칙
- **[NEW]** : TIL 생성 된 지식
- **[PLAN]** : 기간과 계획을 잡고 해야 할 때
- **[POSTING]** : 블로그 포스팅
- **[APPLIYED]** : 지식이 활용 됬을 때

## 나의태도
- 의식적이고 깨어있는 생각으로 항상 더 나은 방법은 없는지 생각하였는가  
- 발생한 문제들 두려워 말고 재밋는 문제 풀이로 생각하였는가

## 출/퇴근 (10점)
- [] 독서를 하고 오늘 읽은 내용 정리 하였는가

## 영어공부 (20점)
- [] 최소 30분, 영어 1문장 씹어먹었는가

## 코딩문제 (15점)
- [] 코딩문제 풀기 연습 이행 여부

## TIL 정리 (25점)
- [] 생각, 에피파니, 개발 지식을 정리 하였는가

## 이번달 목표 진척 체크 (30점)
현재 달에 세운 목표 진척 체크 하기

ex)
[] 패스트 캠프 컴퓨터 공학 강의 - OS 파트 정리 되고 있는가?
[] 리펙토링/디자인 패턴 정리 되고 있는가?
