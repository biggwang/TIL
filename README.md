# :lock: 작성규칙
- 이해한 내용을 **나만의 언어** 로 작성
- 각 doc별로 Learning-Queue.md 파일을 만들어 궁금한거 배우고 싶은거 queue 정리
- 못한것은 주말에 **반드시** 마무리
- 제목에 반드시 TAG 부터 붙인다.

# :pushpin: Tag 작성규칙
- 📕[PLAN]: 기간과 계획을 잡고 해야 할 때
- 📗[POSTING]: 블로그 포스팅
- 📘[APPLIYED]: 지식이 활용 됬을 때
- 🔍[SEARCH]: 추가 조사가 필요 할 때
